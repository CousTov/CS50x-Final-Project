# My Final Project for CS50x 2023

(This was my first time using Flutter x Dart)

### How to build?
Get the dependencies

```bash
flutter pub get
```

Configure project

```bash
flutter create --platforms=android,ios,windows,linux,macos,web .
```

Build
>[!NOTE]
>The path to the application(s) will be echoed in the terminal once it finishes building 

For android
   
```bash
flutter build apk
```
    
For windows

```bash
flutter build windows
```
    
Linux

```bash
flutter build linux
```
    
For MacOS

```bash
flutter build macos
```
    
For Web

```bash
flutter build web
```
    
For IOS

```bash
flutter build ipa
```
